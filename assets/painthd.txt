PaintHD

PaintHD is a monochrome bitmap paint program for the Picocomputer RP6502 target.
It works on a 640x480 canvas in 1-bit color mode and stores images as monochrome BMP files.

Overview

- Left mouse button and right mouse button paint with opposite colors.
- The swap icon changes which mouse button paints white and which paints black.

Toolbar

- The toolbar shows brush size, mouse coordinates, hover hints, and operation status.
- The toolbar can be moved by dragging its handle.
- Double-clicking the toolbar handle moves it to the top-left corner.
- Double-clicking the vertical brush icon toggles between vertical and horizontal orientation.
- Double-clicking the diagonal brush icon flips the diagonal direction between left and right.
- Double-clicking the mirror icon toggles between vertical and horizontal mirror mode.
- Minus and plus change brush size by one step.
- Single click on the size field sets size to 1 px.
- Double-click on the size field sets size to 5 px.
- Triple-click on the size field sets size to the maximum (64 px) size.
- The status field shows operation results, selection state, paste mode, tool changes, and hover descriptions for toolbar icons.

Mouse and keyboard

- LMB paints with the color assigned to the left button.
- RMB paints with the color assigned to the right button.
- Mouse wheel adjusts the brush size, even mid-stroke.
- MMB toggles a full-screen crosshair overlay centered on the cursor position.
- Ctrl sets the status bar to "set start point" while held.
- Ctrl + LMB sets the start point of a straight line; an X marker appears at that position and the status bar changes to "set end point".
- Ctrl + LMB at the end point draws a line with the left-button color.
- Ctrl + RMB at the end point draws a line with the right-button color.
- After drawing, the end point automatically becomes the new start point, enabling continuous polyline drawing while Ctrl is held.
- Releasing Ctrl cancels the current line session and removes the X marker.
- Ctrl + Shift + LMB or Ctrl + Shift + RMB constrains the line to horizontal or vertical segments.
- Ctrl + A clears the current selection and returns to normal painting.
- Ctrl + C copies the current selection to the clipboard file.
- Ctrl + M enters zoom area selection mode; move the pointer to position the zoom area, then click LMB to open the pixel editor.
- Ctrl + Q saves the image to "paintHD_save.bmp" and exit.
- Ctrl + S saves the image to the current save file.
- Ctrl + V starts opaque paste preview.
- Shift + Ctrl + V starts transparent paste preview.
- Ctrl + X copies the current selection and clears the selected area.
- Ctrl + Y performs redo.
- Ctrl + Z performs undo.
- F1 shows the help screen.
- F2 toggles undo on and off.
- F3 saves the current image as an undo checkpoint.
- Esc cancels selection mode, paste preview, primitive preview, zoom mode, and long operations that support cancellation.

Brush tools

Available brush shapes:

- square
- circle
- line vertical/horizontal
- diagonal line L/R line
- spray
- fill

Brush behavior:

- The active brush is used for freehand painting, straight lines, rectangles, and ellipses.
- The fill tool is only available in normal brush mode.
- Rectangle and ellipse tools are disabled when fill is selected.
- Fill is disabled when rectangle or ellipse mode is active.

Primitive tools

Rectangle tool:

- Select the rectangle icon.
- Press and drag on the canvas to define the rectangle.
- Release the mouse button to draw the final shape.
- Hold left Ctrl while sizing to force equal width and height, producing a square.

Ellipse tool:

- Select the ellipse icon.
- Press and drag on the canvas to define the ellipse.
- Release the mouse button to draw the final shape.
- Hold left Ctrl while sizing to force equal width and height, producing a circle.

Selection tool

- Select the selection icon to enter rectangular selection mode.
- Drag with LMB on the canvas to create a selection rectangle.
- While dragging, the current selection size is shown in the status area.
- After the selection is finished, the border is shown as a dotted rectangle.
- Clicking the selection icon again exits selection mode.
- While a selection is active, painting and canvas operations work only inside the selected area.

Clipboard and paste

- The clipboard is stored in a temporary file on disk.
- Opaque paste writes all source pixels.
- Transparent paste writes only set pixels and leaves zero pixels unchanged.
- During paste preview, the preview follows the mouse cursor.
- LMB confirms paste.
- RMB or Esc cancels paste preview.

Canvas operations

- Wipe all clears the full canvas when selection mode is not active.
- Wipe selection clears only the selected area when a selection exists.
- Invert canvas inverts all pixels.
- Invert selection inverts only the selected area.
- Mirror canvas flips the full canvas horizontally or vertically.
- Mirror selection flips only the selected area horizontally or vertically.

Zoom pixel editor

- Ctrl + M enters zoom area selection mode.
- The zoom area marker follows the pointer; click LMB on the canvas to open the pixel editor for that area.
- The pixel editor shows a 32x32 pixel area as 8x8 pixel blocks.
- The toolbar moves to the bottom-left corner and all icons are inactive while the pixel editor is open.
- Click LMB on a block to toggle it; hold LMB and drag to paint multiple blocks with the same value.
- The toolbar coordinate display shows the current block position as [x,y].
- Press Enter to confirm changes and write them to the canvas.
- Press Esc to close the pixel editor without applying changes.
- Entering zoom mode creates an undo checkpoint if undo is enabled.

Undo and redo

- PaintHD keeps up to 255 undo and redo steps on disk.
- New editing operations clear the redo stack.
- Undo and redo affect the canvas contents and restore the dirty state of the document.

Start sequence

- The splash screen is displayed until the user clicks LMB anywhere.
- If argv[1] is a BMP file path, that file is loaded and used as the save target.
- If argv[1] is /new, the canvas is cleared and saved as paintHD_new.bmp.
- If no argument is given and paintHD_save.bmp exists, it is loaded and restored.
- If no argument is given and paintHD_save.bmp does not exist, the canvas is cleared and saved as paintHD_new.bmp.

File behavior

- A BMP file may be loaded from argv[1] on startup.
- The save path may be supplied in argv[2].
- If no input file is supplied, PaintHD can restore painthd_save.bmp automatically after startup.
- The save icon is disabled until the canvas is modified.
- Save and restore status are shown in the toolbar status field.

Notes

- PaintHD is designed for cc65 and the RP6502 target.
- The canvas format is 1 bit per pixel.
- BMP save and load are intended for matching monochrome image data.
